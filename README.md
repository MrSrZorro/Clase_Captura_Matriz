# Instrucciones SeaView
## Alineamiento
### 1.- Descargar Seaview (https://doua.prabi.fr/software/seaview)
### 2.- Abrir matriz en Seaview
### 3.- Para empezar alineamiento ir a ALING, seleccionar metodo y dar ALING ALL

## Concatenar 
### 1.- Abrir SeaView
### 2.- Dar Click en OPEN y abrir una matriz
### 3.- Repetir paso 2 las veces necesaria segun queramos concatenar
### 4.- Dar Click en FILE e posteriormente buscar la opcion CONCATENATE
### 5.- En Source Alingment seleccionar la fuente que usaremos y Target Aligment la que uniremos
### 6.- Finalmente Dar aceptar
